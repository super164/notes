# 一、获取请求头

```go
http://localhost:8090/param?name=zhang&age=18
```

获取请求头信息

```go
package main

import (
	"fmt"
	"net/http"
)

func param(w http.ResponseWriter, r *http.Request) {
	h := r.Header
	fmt.Fprintln(w, h)
	fmt.Fprintln(w, h["Accept-Language"])
}
func main() {
	server := http.Server{Addr: ":8090"}
	http.HandleFunc("/param", param)
	server.ListenAndServe()
}

```

# 二、获取请求参数

- 请求参数可以一次全部获取也可以按照名称获取

```go
package main

import (
	"fmt"
	"net/http"
)

func param(w http.ResponseWriter, r *http.Request) {
	h := r.Header
	fmt.Fprintln(w, h)
	fmt.Fprintln(w, h["Accept-Language"])
	for _, n := range h {
		fmt.Fprintln(w, n)
	}
	r.ParseForm()
	fmt.Fprintln(w, r.Form)
	fmt.Fprintln(w, r.Form["name"][0])
	fmt.Fprintln(w, r.FormValue("name"))
}
func main() {
	server := http.Server{Addr: ":8090"}
	http.HandleFunc("/param", param)
	server.ListenAndServe()
}

```
