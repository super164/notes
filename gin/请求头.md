# 请求头相关

## 请求头参数获取

GetHearder,可以大小写部分，且返回切片中的第一个数据

```go
//请求头的获取方式
router.GET("/", func(context *gin.Context) {
    //首字符不区分大小写，单词之间用“-”连接
    //获取一个请求头
    fmt.Println(context.GetHeader("User-Agent"))
    fmt.Println(context.GetHeader("Accept"))

    //获取全部请求头
    fmt.Println(context.Request.Header)
    //使用Get方法或者.GetHeader可以不用区分大小写，并且返回第一个Value
    fmt.Println(context.Request.Header.Get("User-Agent"))
    //这样取值，需要注意大小写问题
    fmt.Println(context.Request.Header["User-Agent"])

    //自定义的请求头，用Get方法也是免大小写的
    fmt.Println(context.Request.Header.Get("Token"))

    context.JSON(200, gin.H{"username": "futao", "age": 18, "msg": "请求需要"})
})
```





# 响应头相关

## 如何设置响应头

```go
router.GET("/res", func(c *gin.Context) {
    c.Header("Token", "dasfsagasfasdf")
    c.Header("Content-Type", "application/text;charset=utf-8")
    c.JSON(0, gin.H{"data": "查看响应头"})
})
```
