有序集合的相关命令都是以Z开头的

1. ZADD 集合名 分数 成员

   ```redis
   添加一个高考录取分数的有序集合
   ZADD result 680 清华 660 北大 650 复旦 640 浙大
   ```

2. ZRANGE 集合名 起始位置 结束位置：

   查看集合中的元素

3. ZRANGE 集合名 起始位置 结束位置 WITHSCORES:

   在输出集合元素的时候同时输出分数

4. ZSCORE 集合名 元素名：

   输出集合中指定元素的分数

5. ZRANK 集合名 元素名：（从小到大）

   该元素在此有序集合中的排序的位置

6. ZREVRANK  集合名 元素名：（从大到小）

7. ZREM 集合名 元素：

   删除集合中某位元素
